Użyte technologie

html

HyperText Markup Language to język znaczników do tworzenia stron internetowych.

css

Cascading Style Sheets to język służący do opisu wyglądu stron internetowych.

js

JavaScript to język programowania, który umożliwia dodawanie interaktywności do stron internetowych.

Frameworki

Bootstrap

Bootstrap to framework CSS, który ułatwia tworzenie responsywnych stron internetowych.

React

React to biblioteka JavaScript do budowy interfejsów użytkownika.
